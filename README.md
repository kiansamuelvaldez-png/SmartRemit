# SmartRemit Booth

Voice-enabled remittance kiosk powered by Stellar.

## Problem
A construction worker in the Philippines loses time and money using traditional remittance services.

## Solution
SmartRemit Booth enables instant, low-cost transfers using Stellar blockchain.

## Timeline
Week 1: Smart contract  
Week 2: Frontend kiosk UI  
Week 3: Integration + demo  

## Stellar Features
- XLM transfers
- Soroban smart contracts
- Trustlines

## Vision
Fast, accessible remittance for unbanked users.

## Prerequisites
- Rust
- Soroban CLI

## Build
soroban contract build

## Test
cargo test

## Deploy
soroban contract deploy

## Sample Call
send_remittance <from> <to> 1000
